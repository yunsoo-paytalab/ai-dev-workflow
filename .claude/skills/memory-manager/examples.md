# 메모리 파일 예시

## 도메인 목록 예시

### 예시 1: 온라인 교육 플랫폼

| 도메인명   | 설명                   | 주요 리소스                |
| ---------- | ---------------------- | -------------------------- |
| Course     | 강의 관련 핵심 도메인  | 강의, 커리큘럼, 챕터       |
| Student    | 학생 및 학습 진도 관리 | 학생 정보, 수강 기록, 진도 |
| Assessment | 평가 및 시험 관리      | 퀴즈, 시험, 성적           |
| Payment    | 결제 및 구독 관리      | 결제 내역, 구독, 환불      |

### 예시 2: 협업 도구

| 도메인명      | 설명              | 주요 리소스                |
| ------------- | ----------------- | -------------------------- |
| Workspace     | 워크스페이스 관리 | 워크스페이스, 멤버, 권한   |
| Project       | 프로젝트 관리     | 프로젝트, 태스크, 마일스톤 |
| Communication | 커뮤니케이션 도구 | 채팅, 스레드, 알림         |
| File          | 파일 관리 시스템  | 파일, 폴더, 공유 링크      |

## 페이지 구조 예시

### 예시 1: 소셜 미디어 플랫폼

| URL 경로         | 페이지명    | 관련 도메인   | 인증 필요 |
| ---------------- | ----------- | ------------- | --------- |
| /                | 홈          | Feed          | No        |
| /login           | 로그인      | Auth          | No        |
| /signup          | 회원가입    | Auth          | No        |
| /feed            | 피드        | Feed, Post    | Yes       |
| /profile/:userId | 프로필      | User, Post    | No        |
| /post/:postId    | 게시물 상세 | Post, Comment | No        |
| /messages        | 메시지      | Message       | Yes       |
| /settings        | 설정        | User          | Yes       |

### 예시 2: 프로젝트 관리 도구

| URL 경로               | 페이지명      | 관련 도메인     | 인증 필요 |
| ---------------------- | ------------- | --------------- | --------- |
| /                      | 랜딩 페이지   | -               | No        |
| /login                 | 로그인        | Auth            | No        |
| /dashboard             | 대시보드      | Project, Task   | Yes       |
| /projects              | 프로젝트 목록 | Project         | Yes       |
| /projects/:id          | 프로젝트 상세 | Project, Task   | Yes       |
| /projects/:id/board    | 칸반 보드     | Task            | Yes       |
| /projects/:id/timeline | 타임라인      | Task, Milestone | Yes       |
| /team                  | 팀 관리       | Team, User      | Yes       |
| /reports               | 보고서        | Analytics       | Yes       |

## 기능 진행 상황 예시

**상태 설명**:

- ✅ 완료: 단계 완료됨
- 🔄 진행중: 현재 작업 중
- ⏳ 대기: 아직 시작 안함
- - : 해당 없음

### 예시 1: 이커머스 프로젝트

| 기능 ID | 기능명        | 도메인  | 상태      | UI 설계 | 구현 | API 연동 | 비고    |
| ------- | ------------- | ------- | --------- | ------- | ---- | -------- | ------- |
| F001    | 상품 목록     | Product | ✅ 완료   | ✅      | ✅   | ✅       |         |
| F002    | 장바구니 관리 | Cart    | 🔄 진행중 | ✅      | 🔄   | ⏳       | 구현 중 |
| F003    | 사용자 로그인 | User    | ⏳ 대기   | ⏳      | ⏳   | -        |         |

### 예시 2: 블로그 플랫폼

| 기능 ID | 기능명      | 도메인  | 상태      | UI 설계 | 구현 | API 연동 | 비고           |
| ------- | ----------- | ------- | --------- | ------- | ---- | -------- | -------------- |
| F001    | 글 작성     | Post    | ✅ 완료   | ✅      | ✅   | ✅       |                |
| F002    | 댓글 시스템 | Comment | ✅ 완료   | ✅      | ✅   | ✅       |                |
| F003    | 검색 기능   | Search  | 🔄 진행중 | ✅      | 🔄   | ⏳       | 인덱싱 구현 중 |
| F004    | 태그 관리   | Tag     | ⏳ 대기   | ⏳      | ⏳   | -        |                |
