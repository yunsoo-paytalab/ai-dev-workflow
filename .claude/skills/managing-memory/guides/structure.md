## 섹션 설명

### 프로젝트 개요

- 프로젝트 기본 정보
- 초기화 시 자동 생성
- 수동 업데이트 가능

### 현재 진행 상황

- 워크플로우 단계별 진행 상태
- 도메인/Feature별 진행 상태
- **Hook이 자동 업데이트**

### 핵심 결정사항

- 중요한 기술적/아키텍처 결정
- **리드 에이전트가 수동 기록**
- 날짜, 결정 내용, 근거 포함

### 미결 사항

- 해결되지 않은 이슈
- 결정 대기 항목
- 완료 시 체크

### 세션 기록

- 작업 세션 기록
- **Hook이 자동 기록**
- 날짜, 작업 내용, 결과

## 업데이트 규칙

### 자동 업데이트 (Hook)

- 현재 진행 상황
- 세션 기록
- 타임스탬프

### 수동 업데이트 (리드 에이전트)

- 핵심 결정사항
- 미결 사항

## 주의사항

1. **구조 유지** - 섹션 헤더 변경 금지
2. **기존 내용 유지** - 덮어쓰지 않고 추가
3. **간결하게** - 테이블 형식으로 요약
