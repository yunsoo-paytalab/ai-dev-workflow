# 결정사항 기록 가이드

## 기록 프로세스

### Step 1: 메모리 경로 확인

```bash
cat .claude/docs/memory/.memory-ref
```

메모리 파일 위치: `~/.claude-aidev-memory/projects/{id}/memory.md`

### Step 2: 결정사항 기록

**memory.md의 "핵심 결정사항" 섹션에 추가:**

```markdown
## 핵심 결정사항

| 날짜       | 결정 내용             | 근거                  |
| ---------- | --------------------- | --------------------- |
| 2024-01-16 | JWT 인증 방식 채택    | 팀 표준, 확장성       |
| 2024-01-16 | Zustand 상태관리 사용 | 간단한 API, 작은 번들 |
```

### Step 3: 미결 사항 업데이트 (선택)

결정으로 인해 해결된 미결 사항이 있다면:

```markdown
## 미결 사항

- [x] 상태 관리 라이브러리 선정 → Zustand 확정
- [ ] API 연동 레이어 상세 설계
```

## 기록 형식

### 결정 내용

- **간결하게** 1줄로 요약
- 핵심 내용만 포함
- 예: "JWT 인증 방식 채택", "React Query로 서버 상태 관리"

### 근거

- 결정 이유를 간략히
- 2-3개 키워드로
- 예: "팀 표준, 확장성", "캐싱 자동화, 간단한 API"

### 날짜

- **YYYY-MM-DD** 형식
- 예: "2024-01-16"

## 주의사항

1. **기존 내용 유지** - 덮어쓰지 않고 테이블에 행 추가
2. **간결하게** - 1줄 요약 + 근거
3. **선택적 호출** - 중요한 결정사항만 기록
