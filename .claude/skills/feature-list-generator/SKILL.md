# 기능 목록 생성기

## 사용 지침

요구사항에서 기능을 추출하고 도메인별로 분류하여 기능 목록을 생성합니다.

## 입력

- **메모리**: `parsedData` (우선)
- **파일**: `requirements.md` (단독 실행 시)

## 프로세스

### 1. 기능 추출

- 요구사항에서 기능 식별
- 기능별 고유 ID 할당 (F001, F002, ...)

### 2. 기능 분류 및 분석

- 도메인별 기능 그룹핑
- 복잡도 평가 (Low/Medium/High)
- 기능 간 의존성 분석

### 3. 문서 생성

[template.md](template.md) 참조하여 `docs/feature-list.md` 생성:

- 기능 분류 (도메인별)
- 기능 테이블 (ID, 기능명, 설명, 복잡도, 도메인, 상태)
- 기능 의존성 다이어그램 (Mermaid)

## 출력

- `docs/feature-list.md`

## 내부 데이터 전달

- 메모리 변수: `featureList` (page-structure-designer로 전달)

## 생성 범위 제한 ⚠️

**포함할 것**:
✅ 기능 ID 및 기능명
✅ 도메인별 분류
✅ 복잡도 평가
✅ 기능 간 의존성
✅ 의존성 다이어그램 (Mermaid)

**포함하지 말 것**:
❌ UI 화면 구성
❌ API 엔드포인트
❌ DB 테이블 설계
❌ 사용자 스토리 상세
❌ 테스트 케이스
❌ 구현 우선순위

## 가이드라인

- 명확한 기능 단위로 분리
- 적절한 복잡도 평가
- 의존성 명확히 표시
- **간결함 최우선** (템플릿 범위 초과 금지)
