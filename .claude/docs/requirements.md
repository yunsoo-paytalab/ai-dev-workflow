## 영업시간 및 휴무일 페이지 (/business-hours)

### 페이지 구성

4개 섹션으로 구성되며, 각 섹션별 수정 버튼 존재

#### 1. 영업시간 섹션

- 현재 설정된 영업시간 정보 표시
- **표시 규칙**
  - 모든 요일 동일 → "매일"로 표시
  - 요일별 상이 → 요일별 그룹화
- **수정 버튼**: 영업시간 수정 페이지(`/business-hours/edit`)로 이동

#### 2. 정기휴무일 섹션

- 현재 설정된 정기휴무일 정보 표시
- 미설정 시 "설정 안함" 표기
- **수정 버튼**: 정기 휴무일 수정 페이지(`/business-hours/holidays/regular`)로 이동

#### 3. 임시휴무일 섹션

- 현재 설정된 임시휴무일 정보 표시
- 미설정 시 "설정 안함" 표기
- **수정 버튼**: 임시 휴무일 수정 페이지(`/business-hours/holidays/temporary`)로 이동

#### 4. 영업하지 않는 공휴일 섹션

- 영업하지 않는 공휴일 목록 표시
- **수정 버튼**: 영업하지 않는 공휴일 수정 페이지(`/business-hours/holidays/public`)로 이동

## 영업시간 수정 페이지 (/business-hours/edit)

### 상단 탭 구성

- **매일** 탭
- **평일/주말** 탭
- **요일별** 탭

### 탭 진입 조건

- 첫 진입 시: 기존 설정한 탭으로 이동
- 영업시간 미설정 시: 첫 번째 탭(매일)에 포커싱

### 영업시간 표기

- 24시간 기준 (예: 00:00 ~ 24:00)

---

### 요일별 탭

#### 영업 토글 (ON/OFF)

##### 토글 상태 규칙

| 상태 | 동작                                          |
| ---- | --------------------------------------------- |
| ON   | 해당 요일 영업시간 입력 컴포넌트 노출         |
| OFF  | 모든 영업시간 입력 영역 미노출, 인터랙션 불가 |

##### 토글 초기값 (페이지 진입 시)

| 조건                            | 토글 상태 | 영업시간           |
| ------------------------------- | --------- | ------------------ |
| 정기휴무일 존재                 | OFF       | -                  |
| 영업시간 존재                   | ON        | 기존 영업시간 표시 |
| 영업시간 없음 + 정기휴무일 없음 | ON        | null (빈값)        |

##### 토글 전환 동작

| 전환     | 조건                     | 저장 시 동작                                 |
| -------- | ------------------------ | -------------------------------------------- |
| ON → OFF | 영업시간 null (초기상태) | 해당 요일 정기휴무일 등록                    |
| ON → OFF | 영업시간 존재            | 영업시간 삭제 + 해당 요일 정기휴무일 등록    |
| OFF → ON | -                        | 기존 저장된 영업시간 복원 노출 (없으면 빈값) |

##### 정기휴무일 연동 예시

- 요일별 탭: 월요일 Toggle OFF + 저장 → "매주 월요일" 정기휴무 등록
- 평일/주말 탭: 평일 Toggle OFF + 저장 → "매주 월요일", "매주 화요일", "매주 수요일", "매주 목요일", "매주 금요일" 각각 정기휴무 등록

#### 영업시간 Default 값 (탭 전환 시)

| 이전 설정        | 적용 규칙                                       |
| ---------------- | ----------------------------------------------- |
| 요일별 기존 설정 | 동일 적용                                       |
| 평일/주말 설정   | 월~금: 평일 영업시간, 토~일: 주말 영업시간 적용 |
| 매일 설정        | 전체 요일 동일 적용                             |
| 미설정           | 00:00~00:00                                     |

#### 시간 드롭다운

- **시간 단위**: 5분
- **시간 범위**: 00~24
- **분 범위**: 00~55
- **시작 시간을 종료시간 이후로 변경할 경우**: 종료시간 빈값 처리

#### [추가] 버튼 동작

| 조건                                   | 동작                                                                                                        |
| -------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| 시작 & 종료 시간 지정 상태에서 추가    | 시작 시간: 기존 종료 시간 +1시간, 종료 시간: 추가된 시작 시간 +1시간 (예: 10:00~11:00 → 추가 → 12:00~13:00) |
| 시작 or 종료 시간 미지정 상태에서 추가 | 시작 시간: 00:00(빈값), 종료 시간: 00:00(빈값)                                                              |
| 최대 갯수                              | 제한 없음                                                                                                   |

#### 삭제 버튼

- 2번째 행부터 우측에 삭제 버튼 노출
- 클릭 시 해당 행 삭제

---

### 평일/주말 탭

#### 기본 동작

- 요일별 탭과 동일한 동작
- 단, "N요일" 대신 **평일, 토요일, 일요일**로 노출

#### 영업시간 Default 값 (탭 전환 시)

| 이전 설정           | 적용 규칙                                                     |
| ------------------- | ------------------------------------------------------------- |
| 평일/주말 기존 설정 | 동일 적용                                                     |
| 요일별 설정         | 평일: 월요일(없으면 화>수>목>금), 토: 토요일, 일: 일요일 적용 |
| 매일 설정           | 평일/주말 모두 '매일' 영업시간 적용                           |
| 미설정              | 00:00~00:00                                                   |

---

### 매일 탭

#### 영업시간 Default 값 (탭 전환 시)

| 이전 설정      | 적용 규칙                                                    |
| -------------- | ------------------------------------------------------------ |
| 매일 기존 설정 | 동일 적용                                                    |
| 평일/주말 설정 | 평일 영업시간 적용 (평일 없을 경우 주말 영업시간 적용)       |
| 요일별 설정    | 월요일 영업시간 적용 (월요일 없을 경우 화>수>목>금>토>일 순) |
| 미설정         | 00:00~00:00                                                  |

---

### 공휴일 영업시간 (공통)

- **위치**: 모든 탭(매일, 평일/주말, 요일별)의 모든 요일 설정 하단에 노출
- **Default**: 기존 설정된 공휴일 영업시간 적용, 미설정 시 00:00
- **시간 변경**: 요일별 영업시간 변경 기능과 동일

### 정기 휴무일 추가

#### [추가] 버튼

- 클릭 시 "정기 휴무일 추가" 바텀시트 오픈

#### 정기 휴무일 추가 바텀시트

##### 탭 구성

| 탭   | 설명                          |
| ---- | ----------------------------- |
| 매주 | Default 탭, 요일 선택 UI 노출 |
| 매월 | 주차 선택 + 요일 선택 UI 노출 |

- **탭 이동 시**: 기존 선택값 초기화

##### [매주] 탭

- **요일 선택**: 월, 화, 수, 목, 금, 토, 일 (복수 선택 가능)
- **선택 예시**: 화, 목 선택 시 → "매주 화요일", "매주 목요일" 각각 추가
- **중복 처리**: 이미 추가된 요일도 선택 가능하나, 재추가 시 목록에 중복 추가되지 않음

##### [매월] 탭

- **주차 선택**: 1주차, 2주차, 3주차, 4주차, 5주차 (복수 선택 가능)
- **요일 선택**: 월, 화, 수, 목, 금, 토, 일 (복수 선택 가능, 매주 탭과 동일 UI)

##### [추가] 버튼

| 조건             | 동작                                               |
| ---------------- | -------------------------------------------------- |
| 선택한 사항 없음 | 비활성화                                           |
| 선택한 사항 있음 | 활성화 → 클릭 시 컨펌창 노출 → 확인 시 휴무일 추가 |

---

### 등록된 정기 휴무일 목록

#### 표시 형식

| 휴무 타입 | 표시 형식           | 예시              |
| --------- | ------------------- | ----------------- |
| 매주      | 매주 [요일]         | 매주 화요일       |
| 주차별    | 매월 [N주차] [요일] | 매월 1주차 수요일 |

#### 목록 정렬 순서

1. **타입 우선**: 매주 → 주차별
2. **요일 순서**: 월 → 화 → 수 → 목 → 금 → 토 → 일

#### 삭제 버튼

- 각 정기 휴무일 셀 우측에 삭제 버튼 노출
- **[삭제] 버튼 클릭 시**: 정기 휴무일 삭제 모달 노출

---

### 정기 휴무일 삭제 모달

#### 모달 구성

- 삭제할 정기휴무일의 영업시간 설정 UI
- [설정 완료 및 삭제] 버튼

#### 동작 Flow

1. 삭제 버튼 클릭 → 삭제 모달 오픈
2. 해당 요일의 영업시간 설정
3. [설정 완료 및 삭제] 버튼 클릭
4. 정기휴무일 삭제 + 해당 요일 영업시간 업데이트

#### [설정 완료 및 삭제] 버튼

| 조건               | 동작     |
| ------------------ | -------- |
| 영업시간 미설정    | 비활성화 |
| 영업시간 설정 완료 | 활성화   |

#### 예시

- 매주 월요일 정기휴무 삭제 시:
  1. 모달 내에서 월요일 영업시간 설정 완료
  2. [설정 완료 및 삭제] 버튼 클릭
  3. 정기휴무일(매주 월요일) 삭제
  4. 월요일 영업시간 업데이트

### 임시 휴무일과 공휴일 데이터 관계

> **중요**: 임시 휴무일과 공휴일은 서버 데이터 상 동일한 데이터에 속함

- 임시 휴무일 등록 시 스케줄 정보와 함께 공휴일 정보를 불러옴
- 선택한 기간 내 공휴일이 포함된 경우, 로직에서 자동으로 감지하여 분리 등록
- 분리 등록 시 별도 API를 각각 호출하여 처리
- 사용자에게 별도 모달 미노출 (자동 처리)

#### 공휴일 분리 등록 예시

**사용자 선택**: 12월 22일 ~ 12월 27일 (크리스마스 12/25 포함)

**등록 결과** (3건 분리 등록):

1. 25.12.22 ~ 25.12.24 (임시휴무)
2. 25.12.25 (크리스마스) - 공휴일로 등록
3. 25.12.26 ~ 25.12.27 (임시휴무)

### 임시 휴무일 추가

#### [추가] 버튼

- 클릭 시 "임시 휴무일 추가" 바텀시트 오픈 (날짜 선택 캘린더)

#### 임시 휴무일 추가 바텀시트

##### 날짜 선택 (캘린더)

- 기존 캘린더 컴포넌트 활용
- 기간 선택 가능 (시작일 ~ 종료일)

##### 캘린더 내 날짜 상태

| 상태            | 표시          | 동작                   |
| --------------- | ------------- | ---------------------- |
| 기존 임시휴무일 | Disabled 표시 | 선택 가능 (11.25 변경) |
| 정기휴무일      | 일반 표시     | 선택 가능              |
| 일반 날짜       | 일반 표시     | 선택 가능              |

##### 날짜 선택 제한

| 조건                                     | 동작                                                |
| ---------------------------------------- | --------------------------------------------------- |
| 7일 초과 기간 선택 시                    | 지정 불가 Toast 팝업 + 시작일로부터 7일로 자동 지정 |
| 복수 구간 선택 시 (비연속 2개 이상 구간) | 지정 불가 Toast 팝업 + 선택 불가                    |

##### [추가하기] 버튼

| 조건        | 동작                         |
| ----------- | ---------------------------- |
| 날짜 미선택 | 비활성화                     |
| 날짜 선택됨 | 활성화 → 클릭 시 컨펌창 노출 |

##### 추가 완료 시

- 바텀시트 닫기
- Toast 팝업: "임시 휴무일이 추가되었어요"
- **기존 임시휴무일 재등록 시**: 덮어쓰기 처리 (schedule API 호출 시 기존 임시휴무일 업데이트)

### [반려하기] 버튼

#### 동작 Flow

## 데이터 정의 및 제약사항

### 영업시간 설정 기본값

- 설정된 영업시간이 없을 경우: 00:00~00:00

### 시간 드롭다운 설정

- 시간 단위: 5분
- 시간 범위: 00~24시
- 분 범위: 00~55분

### 임시 휴무일 제약

- 최대 연속 7일까지만 선택 가능 (초과 시 시작일로부터 7일로 자동 지정)
- 한 번에 하나의 연속 구간만 선택 가능 (복수 구간 선택 불가)
- 정기휴무일 및 기존 임시휴무일과 중복 등록 가능
- 기존 임시휴무일 재등록 시 덮어쓰기 처리

### 정기 휴무일 제약

- 매주 정기 휴무: 월~일 모든 요일 선택 가능, 복수 선택 가능
- 주차별 정기 휴무: 1~5주차, 모든 요일 선택 가능, 복수 선택 가능
- 이미 등록된 정기휴무일 재추가 시 중복 추가되지 않음

### 정기휴무일과 영업시간 토글 연동

- 정기휴무일로 설정된 요일은 영업시간 수정 페이지에서 토글 OFF 상태로 표시
- 영업시간 수정 페이지에서 토글 ON → OFF 변경 후 저장 시:
  - 영업시간 null 상태: 해당 요일 정기휴무일 등록
  - 영업시간 존재 상태: 영업시간 삭제 + 해당 요일 정기휴무일 등록
- 저장 시 "변경사항을 저장하시겠습니까?" 모달 노출 (별도 안내 모달 없음)

### 임시휴무일과 공휴일 데이터 연동

- 임시휴무일과 공휴일은 서버에서 동일한 데이터로 관리
- 임시휴무일 등록 시 선택 기간 내 공휴일이 포함된 경우 자동 분리 등록
- 임시휴무일 목록과 영업하지 않는 공휴일 목록은 동일 데이터를 공유
- 한쪽에서 삭제 시 다른 쪽에서도 삭제됨
