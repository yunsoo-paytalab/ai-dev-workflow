# ACCOUNT-002: 계정 목록 조회 및 복사

## 기본 정보

- **카테고리**: 계정 관리
- **상태**: 설계 중
- **관련 페이지**: `/account/accounts`
- **권한**: QA Tool 사용자

## 개요

검색된 계정 목록을 테이블 형태로 표시하고 식별자를 복사할 수 있습니다.

## 주요 요구사항

- 회원 식별자, 이름, 전화번호, 구분, 상태, 주문 상태, 생성일 표시
- 복사 버튼 제공 (회원 식별자)
- 탈퇴한 회원의 경우 탈퇴일 표시

## UI/UX 구성

**주요 컴포넌트**:

- 테이블 (회원 식별자, 이름, 전화번호, 구분, 상태, 주문 상태, 생성일, 관리)
- 복사 버튼

**인터랙션**:

- 복사 버튼 클릭 시 클립보드에 복사
- 복사 성공 토스트

## 데이터 연동

**API 엔드포인트**:

- `GET /api/accounts?phoneNumber={phoneNumber}` - 계정 목록 조회

**상태 관리**:

- 로컬: 계정 목록

## 관련 도메인 인터페이스

**주요 엔티티**:

- `PasserAccount` - 회원/비회원 계정 (Union)
- `UserAccount` - 회원 계정
- `NonUserAccount` - 비회원 계정

## 의존성

**선행 기능**: ACCOUNT-001 (전화번호로 계정 검색)
**후속 기능**: ACCOUNT-003 (회원 탈퇴), ACCOUNT-004 (비회원 삭제)
